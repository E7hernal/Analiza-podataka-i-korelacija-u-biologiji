This repository containts code and text for "Analiza podataka i korelacija u biologiji" course (5th year, Physics Major).

schlogl.ipynb contains the code simulating a chemical reaction (i.e. Schl√∂gl model) via Stochastic Simulation Algorithm (SSA), also known as Gillespie algorithm. Running time is around 30 mins with current parameters.

SIRS.ipynb contains markdown overview of linear stability analysis for the SIRS disease model.
